Overview

Users can start conversations on the main navigation bar on all interfaces at the top-right corner. Users are able to create group conversations with the different boards they have on Trello as well as with individuals within any of their boards.

At any time during a conversations, users are able to save any of the conversation's messages and add them to an existing or new list on their project board.

By holding a finger on a conversation, the messages present on the screen are converted into cards (list items) in a temporary list. This temporary list is then placed alongside existing lists on the project board.

Users can then drag and drop one or multiple messages into another list as an official card.

Once messages have been dragged and dropped, users will be informed that cards were saved and then brought back to their conversation. Users can repeat this process to save other messages.
